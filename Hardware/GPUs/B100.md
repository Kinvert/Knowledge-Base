# B100

**B100** is NVIDIA's entry-level Blackwell architecture datacenter GPU, released in 2024. It was the first Blackwell GPU to ship and is designed as a drop-in replacement for [[H100]] in existing HGX systems - same 700W TDP, same baseboard compatibility. The B100 delivers ~77% more FP16/BF16 throughput than H100 while introducing FP4 support for next-generation inference workloads.

---

## Overview

The B100 targets AI training, inference, and HPC workloads. Key innovations include:

- **5th Gen Tensor Cores:** FP4, FP6, FP8, FP16, BF16, TF32, FP64 support
- **Dual-Die Design:** Two GB100 dies connected via NV-HBI (10 TB/s)
- **HBM3e Memory:** 192GB with 8.0 TB/s bandwidth
- **NVLink 5.0:** 1,800 GB/s GPU-to-GPU bandwidth
- **Drop-in Replacement:** Fits existing HGX H100/H200 baseboards

## Architecture Highlights

- **GB100 GPU:** 208B transistors, dual-die design, TSMC 4nm process
- **NV-HBI:** 10 TB/s inter-die bandwidth connecting two GB100 dies
- **5th Gen Tensor Cores:** First with FP4 support for efficient inference
- **NVLink 5.0:** 18 links x 100 GB/s = 1,800 GB/s bidirectional
- **PCIe Gen6:** 256 GB/s host interface
- **Drop-in Compatible:** Same 700W envelope as H100 SXM

## Strengths

- Drop-in replacement for H100 (same 700W, same baseboard)
- 2.4x memory bandwidth vs H100 (8.0 vs 3.35 TB/s)
- 2.4x VRAM vs H100 (192GB vs 80GB)
- First GPU with FP4 Tensor Core support
- NVLink 5.0 doubles interconnect bandwidth

---

## Weaknesses

- Slightly lower FP16/FP8 compute than H100 (constrained to 700W envelope)
- SXM-only (no PCIe option)
- Limited market availability (most providers chose B200)
- Being overshadowed by B200 for new deployments

## Summary

B100 is NVIDIA's entry-level Blackwell GPU, designed as a drop-in replacement for H100 in existing HGX infrastructure. While it has slightly lower FP16/FP8 compute than H100 (constrained to 700W), it offers 2.4x memory bandwidth, 2.4x VRAM, and introduces FP4 support. Limited market availability as most providers chose B200 instead. For complete build configurations, see [[B100 Builds]].
